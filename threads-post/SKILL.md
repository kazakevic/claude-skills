# Threads Post Creator

## Platform Limits
- 500 characters (10,000 via text attachments)
- ONE tag per post (no # symbol, phrases allowed)
- Up to 20 photos/videos per post

## Post Structure Template
```
[Hook - under 10 words, attention-grabbing]

[Empty line]

[Value/insight - 1-3 sentences]

[Empty line]

[Question or CTA to spark replies]
```

## Algorithm Rules

### ✅ Algorithm LOVES
- Early engagement (first 30 minutes critical)
- Meaningful replies and conversation depth
- Original content and perspectives
- Consistent daily posting
- Images/videos (75% of posts are text-only)
- Quick replies to comments

### ❌ Algorithm PENALIZES
- Engagement bait ("Like if you agree!")
- Rage bait (provoke anger)
- "Follow for more!" prompts
- Copied/unoriginal content
- Post and ghost (no engagement)
- Links without context

### Hook DON'Ts
- Generic openings: "Threads is interesting"
- Weak framing: "Some thoughts on AI..."
- Starting with "I think..."
- Long paragraph openers

### Hook DOs
- Short, punchy first line (under 10 words)
- Spark emotion (curiosity, controversy, recognition)
- Speak directly to target audience
- Use specific numbers
- Leave slight mystery

## Content Frameworks

### "This vs That" Debate
```
"[Tool A] vs [Tool B]

I'll go first: [Your preference]

What's your pick and why?"
```

### Tool Discovery
```
"Tool I discovered this week that's criminally underrated:

[Tool name]

What it does: [Brief explanation]
Why it's good: [Specific benefit]

What's yours?"
```

### "I Built This"
```
"Just shipped [project] using [approach]:

Stack: [tools used]
Time: [how long]
AI usage: [how much vibe coded]

Thread on how I did it 👇"
```

### Learning Journey
```
"Month [X] of learning [skill]:

What's working: [insight]
What's hard: [challenge]
What's next: [goal]

Anyone else on this journey?"
```

### Fill-in-the-Blank
```
"Complete this sentence:

The most overrated AI tool in 2026 is _____

I'll go first: [Your answer]"
```

### Prediction
```
"My prediction for [topic] by end of 2026:

[Bold prediction]

Agree? Disagree? What's yours?"
```
